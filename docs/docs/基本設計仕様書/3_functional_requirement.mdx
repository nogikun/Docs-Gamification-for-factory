## 3.1 求職者向け機能（ゲームを除く）

### 登録・検索
    - アプリに登録されている職業体験情報を、`日時`、`場所`、`業務内容`などの条件で検索できる機能
    - 希望条件に合致する職業体験の自動提案できる機能（アプリを開いた際、上から順に整列させる）
### 応募・通知
    - 興味のある職業体験に対して、アプリ経由で申し込みが行える機能
    - 企業からの受け入れ決定通知を受信し、アプリから通知
### 報酬支払い
    - 職業体験実施後、企業からの報酬支払いの管理
    - 企業からの評価値に応じた`報酬（ブキ）`の受け取り
### 評価
    - インターンシップ体験後、体験した企業への評価（スコアおよびコメント）を入力

## 3.2 企業向け機能

### 職業体験情報の登録
    - カレンダー機能を用いた、インターンシップの`日時`、`業務内容`、`報酬`、`募集人数`、`必要資格`などの情報登録
### 応募者管理
    - 求職者からの申し込みを一覧表示し、受け入れの検討および通知を実施
### 評価
    - インターンシップ実施後、求職者の`評価（スコアおよびコメント）`の入力・管理
### 過去インターン情報の確認
    - 過去のインターンシップ実績や評価履歴を確認可能

## 3.3 アプリ機能（共通・仲介機能）

### マッチング
    - 求職者と企業間の職業体験マッチングをサポートする
### 通知
    - 各プロセス（申し込み、受け入れ決定、支払い完了、評価依頼）に対するリアルタイム通知を行う

## 3.4 ゲーム機能（求職者向け）
### 育成
    - ゲームを通じて、求職者が楽しみながら職業体験を学べる機能（未定）
    - ゲームを通じて、求職者が`報酬（ブキ）`を獲得できる。 
        - `報酬（ブキ）`は、職業体験を通じて得たスキルや知識を代表したもので企業１社につき１つ獲得できる。
        - `ブキ`は戦闘時に回数制限を設け、使い切ると再度職業体験を行う必要がある。
    - 戦闘に勝利すると、経験値を獲得できる。レベルはランキングに反映される。

### ランキング
    - ゲーム内での戦闘結果に応じたランキングを表示
    - ランキングは、１ヶ月ごとにリセットされる。
        - 新規参入者が職業体験を頑張れるモチベーションを提供し続ける
    - ランキングは、`経験値（レベル）`、`ブキの獲得数`など様々な尺度のものがある。

### 戦闘
    - ゲーム内での戦闘機能を提供
    - 戦闘は、求職者が獲得した`ブキ`を使用して行う
    - ターン制で行われ、勝利条件（敵を倒す）を満たすと経験値を獲得できる
    - １日に１回戦闘が可能である。職業体験に参加することで、戦闘回数が増える。
